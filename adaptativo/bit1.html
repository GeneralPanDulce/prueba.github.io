JavaScript

Que es HTML?

El lenguaje de marcado de hipertexto, que facilita el acceso a información no lineal a través de nodos o enlaces, constituye el elemento fundamental de la web. Además de definir la estructura del contenido, también le otorga significado. Junto con HTML, suelen emplearse otras tecnologías para definir la apariencia o presentación de una página web (CSS) y su funcionalidad o comportamiento (JavaScript). Este código es esencial para estructurar y mostrar tanto la página web como su contenido.

DOM

Que es CSS?

CSS es un lenguaje utilizado para dar diseño gráfico y definir la presentación de un documento estructurado escrito en un lenguaje de marcado. Funciona mediante reglas que cada usuario puede definir según sus necesidades de estilo para el documento específico. Con CSS, puedes establecer directrices sobre cómo quieres que se muestre la información en tu sitio web, manteniendo los comandos de estilo separados de aquellos que configuran el contenido.

Que es "A"?

Es un elemento que facilita la creación de enlaces hacia páginas web. Su atributo principal es href, donde especificamos la dirección a la que queremos enlazar. Otro atributo comúnmente utilizado es target, que determina si el enlace se abrirá en una nueva ventana o en la misma. Esta etiqueta permite vincular a otros archivos, páginas web, direcciones de correo electrónico, documentos internos en la misma página o cualquier otra URL.

Propiedad Style

Que es el texto?

Principalmente, el contenido HTML está compuesto por encabezados y párrafos, lo que simplifica la lectura y aumenta el disfrute sin importar el tipo de texto que se esté leyendo. Una de las funciones fundamentales de HTML es proporcionar estructura y significado al texto para que el navegador lo muestre correctamente. Los encabezados van del elemento h1 al h6, cada uno con un propósito diferente: h1 para títulos, h2 para subtítulos, y así sucesivamente. Los párrafos se delimitan con el elemento < p >< /p > y constituyen la base del texto. Además, hay listas que se dividen en ordenadas (con un orden específico, y si no se sigue, afecta el resultado) y no ordenadas (donde el orden no es relevante).

Classlist

Que es "IMG"?

Se refiere a un elemento vacío, lo que implica que no contiene texto ni etiqueta de cierre, y requiere al menos un atributo para ser utilizado, como src (a veces conocido como source). Para incorporar una imagen simple en una página web, se emplea el elemento < img >. Esta etiqueta permite la visualización de imágenes y, en caso de que el archivo esté dañado, también puede describir el gráfico mediante ciertos atributos. Esto resulta útil para ordenadores con navegadores no gráficos, como los utilizados por personas con discapacidad visual.
